    1. 1.1. 形参：在定义函数阶段定义的参数称之为形式参数，简称形参,相当于变量名
    2. 1.2. 实参:在调用阶段传入的值称之为实际参数，简称实参，相当于变量值
    3. 1.3. 实参是传入的值，但值可以是以下形式
    4. 1.4. 形参与实参的具体使用
  2. 2. 关键字实参数：在函数调用阶段，按照 key=value 的形式传入的值
    1. 2.1. 指名道姓给某个参数传值，可以完全不参照顺序
  3. 3. 强调：如果需要混合使用
    1. 3.1. 位置实参必须放在关键字实参前
  4. 4. 默认形参：在定义函数阶段，就可以被赋值的形参，称之为默认参数
    1. 4.1. 特点：在定义阶段就已经被赋值，意味着在调用阶段可以不用为其赋值
    2. 4.2. 默认参数的值是在函数定义阶段被赋值的，准确地说被赋予的是值的内存地址
